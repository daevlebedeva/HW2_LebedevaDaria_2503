# HW2_LebedevaDaria_2503
Учебный проект по ООП на Python. Программа помогает работать с рецептами и автоматически составлять список покупок, хранить ингредиенты, объединять рецепты.

## Установка
git clone <репозиторий>\
cd <папка>\
pip install -r requirements.txt

## Использование

Для использование нужно импортировать файлы проекта в ваш проект\
пример:\

from Ingredient import Ingredient\
banana = Ingredient("Бананчики", 267, "г")
